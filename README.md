# Git Better

>This is where you will find materials for "Git Better" for lab meeting 3/29/17.

## Requirements

* Git
* GitKraken >= v2.2
* A text editor, preferably one with markdown syntax highlighting and/or rendering (e.g., atom.io)
* A GitHub account
